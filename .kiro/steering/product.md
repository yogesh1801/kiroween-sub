# Necromancer Code Resurrector

A horror-themed legacy code translation tool powered by Google Gemini AI. Users can transform "dead" legacy code (COBOL, Fortran, Assembly, BASIC) into modern languages (TypeScript, Rust, Go, Python) through an immersive occult-themed experience.

## Core Features

- **Code Resurrection**: Translate legacy code to modern languages
- **Autopsy Mode**: Analyze and explain legacy code's logic and flaws
- **Curse Removal**: Security fixes, refactoring, and optimization
- **Soul Binding**: Generate documentation and unit tests
- **Full Ritual**: Complete pipeline combining all modes
- **Spirit Medium**: AI chat interface for code Q&A
- **Graveyard**: History of past translations (localStorage)

## Theme & Aesthetic

Dark horror aesthetic with:
- Blood-red and rot-green color schemes
- CRT scanlines and chromatic aberration effects
- Audio effects (whispers, heartbeats, ambient drones)
- Sanity mechanics that affect visual presentation
- Demonic TTS for reading results aloud
